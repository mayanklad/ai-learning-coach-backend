# AI Learning Coach Backend

**Description:**  
This repository contains the backend for the AI-driven Personal Learning Coach. It is built with **FastAPI**, integrates with **OpenAI GPT API** to dynamically generate tasks, and uses **PostgreSQL** for storing user profiles, skills, and task history.

## Features

- User profile and skill management
- Dynamic task generation using GPT API
- Recommendation engine for weakest skills
- Task completion tracking and skill proficiency updates
- PostgreSQL database integration

## Tech Stack

- **Backend:** Python + FastAPI
- **Database:** PostgreSQL + SQLAlchemy
- **AI:** OpenAI GPT API
